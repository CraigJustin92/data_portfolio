Overview:

This project explores the Netflix Movies and TV Shows dataset using Python to build predictive models that identify movies likely to be fast-added (within the first year of release) to Netflix. The goal is to support content acquisition strategies, maximize portfolio value, and enhance viewer engagement.

Approach

  • Data Exploration: Analyze features such as genre, release year, and duration to uncover patterns in fast-added movies.
  • Data Cleaning: Handle missing values, correct inconsistencies, and remove duplicates.
  • Feature Engineering & Selection: Create and refine features aligned with Netflixâ€™s strategic objectives.
  • Hypothesis Testing: Use chi-squared tests to evaluate associations between categorical variables.
  • Model Development & Evaluation: Train and assess multiple machine learning models using accuracy, precision, recall, and F1 score.
  • Model Interpretation: Interpret results to identify key predictors and generate actionable insights.

Focus

Special emphasis is placed on identifying a predictor variable that significantly influences the likelihood of a movie being fast-added to the platform.

Summary of Results

  • Random Forest delivered the best performance (AUC 77.1%, F1 75.1%), outperforming Decision Trees and XGBoost.
  • Mature-rated content (TV-MA, R) was the strongest predictor of fast-added titles.
  • Recommended improvements: collect richer engagement, social media, and regional data, and engineer new interaction features for stronger predictive power.
